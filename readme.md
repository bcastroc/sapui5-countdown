# myUI5App

## API and sample code 
* https://sapui5.hana.ondemand.com/

## Getting started
### Basic UI5 - use easy-ui5 generator
* https://www.npmjs.com/package/generator-easy-ui5
* npm install -g yo generator-easy-ui5
* Scaffold project -> $ yo easy-ui5 project

## Run project
To run project run: 
$ npm start
